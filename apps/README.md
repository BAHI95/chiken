# Apps Workspace

هذا المجلد يجمع تطبيقات الهاتف بشكل منفصل داخل نفس المشروع:

- `android/` لتطبيق Android
- `ios/` لتطبيق iPhone / iPad

كلا التطبيقين يستعملان واجهة مشتركة من `packages/mobile-shell` وثوابت مشتركة من `packages/core`.

هذا الأسلوب أكثر احترافية من تكرار كل شيء مرتين، ويعطيك:
- فصلًا واضحًا لكل منصة.
- قابلية صيانة أفضل.
- إمكانية تخصيص كل منصة لاحقًا بدون كسر المنطق المشترك.
