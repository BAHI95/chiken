# نظام إدارة مزرعة الدواجن

نسخة ويب عربية RTL لإدارة القطيع، البيض، العلف، التفقيس، السجل الصحي، التنظيف، والمالية، مع جاهزية PWA للتثبيت على الهاتف، وواجهة حسابات سحابية قابلة للترحيل إلى بيئة إنتاجية حقيقية عبر Postgres وAPI مستقل.

## ما الذي تعمل عليه هذه النسخة الآن
- واجهة تشغيل يومية متكاملة للمزرعة.
- تخزين محلي آمن نسبيًا مع نسخ احتياطية واسترجاع.
- تنبيهات ومهام يومية وتقويم تشغيلي.
- سجل نشاط وسجل أخطاء داخلي.
- تسجيل وإنشاء حسابات حقيقية عبر API مستقل مع فصل بيانات كل مزرعة.
- صفحة `المنتج والاشتراك` لتقييم النموذج التجاري والنشر.
- دعم PWA للتثبيت والعمل الجزئي دون إنترنت.
- مساحة `apps/` لتطبيق Android وiOS بشكل منفصل مع نواة واجهات مشتركة.
- دعم التحقق بالبريد وإعادة تعيين كلمة المرور.
- ملفات نشر وإعدادات CI/CD للويب والـ API.

## ما الخطوة التجارية التالية
هذه النسخة ممتازة كـ:
- عرض تجريبي للمربين المحليين.
- Pilot ميداني مع عدد محدود من العملاء.
- قاعدة UX ووظائف قبل بناء المنصة السحابية.

أما قبل البيع على نطاق واسع أو بنظام اشتراك، فالخطوات التالية التشغيلية هي:
- ضبط `DATABASE_URL` و`SESSION_SECRET` وبقية الأسرار.
- نشر الويب والـ API وربط الدومين.
- تفعيل بريد الإنتاج عبر Resend.
- إضافة سياسات الخصوصية وحذف الحساب وتصدير البيانات.

## تشغيل المشروع محليًا
للتشغيل المحلي الإنتاجي:
- انسخ `.env.example` إلى `.env`
- اضبط `DATABASE_URL`
- شغّل migrations:
  - `npm run api:migrate`
- شغّل:
  - `npm run dev:local-stack`
